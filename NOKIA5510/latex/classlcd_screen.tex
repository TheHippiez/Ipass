\subsection{Detailed Description}
\hyperlink{classlcd_screen}{lcd\+Screen} class 

This class is used to initialize 5510 screens. It may also be used by any similair L\+CD screen with the same pins and workings as the Nokia 5510 ( Screen sizes might require some changes in the libary) . The following is a short summary of the 6 functions this class provides\+: \hyperlink{classlcd_screen_a613df6d4702cd86941d3a473a846d737}{clear()}; \+: A function that sets every pixel on the screen to zero. \hyperlink{classlcd_screen_a37339d0d8ae50a6efcbae10d3f8c7573}{setup()}; \+: A function that issues commands to the L\+CD screen, setting it up to receive data. \hyperlink{classlcd_screen_ab75ed961943609fbe7f437e9e86abe4d}{command()}; \+: A function that sends the array to the L\+CD screen. \hyperlink{classlcd_screen_a9d8cee1ed44907c5fbfc52bbe1205853}{draw()}; \+: A function that draws a \mbox{[}8\mbox{]}\mbox{[}84\mbox{]} matrix on the screen! \hyperlink{classlcd_screen_af9d494e72329e3eca29209bfd2d05882}{draw\+B\+I\+G()}; \+: A function that draws a \mbox{[}48\mbox{]}\mbox{[}84\mbox{]} matrix on the screen!.
